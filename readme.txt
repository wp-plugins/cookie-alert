Useful for adding a fixed half transparent banner, fullwidth to all of your pages, with custom text and a button to close it.

== Description ==

 Originally designed to add the Cookie Policy advise as requested by recent laws, but you can use it with other intents. 
 
Once installed and activated you will have a new link under "Settings" in your admin panel.
 
In this section you can set the message you what to show, a link url and a link text (both optional) for the privacy/cookie policy that will be append at the end of the message. Then the text for the close button. Last you can on/off it. 

When the plugin is on on frontend in every page of your wordpress install you have the message previously settes in admin panel. The user can hide it clicking on the closing button. By clicking the closing button the user set a cookie that lasts for one day.


== Installation ==

First Option

1. Check for Cookie Alert in Plugin section in your admin panel and install it.
2. Activate the plugin through the 'Plugins' menu in WordPress.

Second option

1. Download cookie-alert.zip from Wordpress repository and unzip it 
2. Upload `cookie-alert` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
